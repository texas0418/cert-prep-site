Question 1 · Calc — Corrosion rate

A pressure vessel shell had an original thickness of 0.5 in. A thickness reading 8 years later is 0.38 in. What is the long-term corrosion rate?

Answer: 0.015 in/yr
Long-term corrosion rate = (t_initial − t_actual) / time = (0.5 − 0.38) / 8 = 0.015 in/yr.
Reference: API 510 7.1
Question 2 · Calc — Remaining life

A vessel component measures t_actual = 0.360 in., with t_required = 0.150 in. and a corrosion rate of 0.012 in/yr. What is the remaining life?

Answer: 17.5 years
Remaining life = (t_actual - t_required) / corrosion rate = (0.360 - 0.150) / 0.012 = 17.5 years.
Reference: API 510 7.2
Question 3 · Calc — Remaining corrosion allowance

A component has an actual thickness of 0.445 in. and a required thickness of 0.3 in. What is the remaining corrosion allowance?

Answer: 0.145 in
Remaining corrosion allowance = t_actual − t_required = 0.445 − 0.3 = 0.145 in.
Reference: API 510 7.1
Question 4 · Calc — Inspection interval

The remaining life of a vessel is calculated to be 12 years. Per API 510, the internal/on-stream inspection interval shall not exceed which value (assuming this governs and no RBI assessment is used)?

Answer: 6 years
API 510 limits the internal or on-stream inspection interval to the lesser of one-half the remaining life or 10 years. One-half of 12 = 6 years, which is less than 10, so 6 years governs.
Reference: API 510 6.5
Question 5 · Calc — Future corrosion allowance

A shell measures 0.5 in. with a corrosion rate of 0.01 in/yr. For an MAWP assessment covering the next 10-year interval, what thickness should be used (deduct the corrosion expected before the next inspection)?

Answer: 0.4 in
t_used = t_measured − (corrosion rate × interval) = 0.5 − (0.01 × 10) = 0.4 in.
Reference: API 510 7.5
Question 6 · Calc — Static head

A vessel part is located 35 ft below the liquid surface. Using the exam convention (specific gravity = 1.0), what is the static head pressure acting on that part?

Answer: 15.15 psi
Static head = 0.433 psi/ft × height × SG = 0.433 × 35 × 1.0 = 15.15 psi.
Reference: API 510 BOK I.A.3
Question 7 · Calc — Internal pressure (shell)

Determine the required thickness of a cylindrical shell for internal pressure (circumferential stress). P = 300 psi, inside radius R = 30 in., S = 17500 psi, E = 1.0. Use t = PR/(SE − 0.6P).

Answer: 0.5196 in
t = PR/(SE − 0.6P) = (300 × 30) / ((17500 × 1.0) − 0.6 × 300) = 0.5196 in.
Reference: ASME VIII-1 UG-27(c)(1)
Question 8 · Calc — Internal pressure (head)

Determine the required thickness of a hemispherical head: P = 400 psi, inside radius R = 30 in., S = 17500 psi, E = 1. Use t = PR/(2SE - 0.2P).

Answer: 0.3436 in
t = PR/(2SE - 0.2P) = (400 x 30) / ((2 x 17500 x 1) - 0.2 x 400) = 0.3436 in. Using the cylinder formula (0.6953 in) is the common error; a hemi-head needs roughly half the thickness of a cylinder of equal radius.
Reference: ASME VIII-1 UG-27(d) / UG-32(f)
Question 9 · Calc — Joint efficiency

A Type 1 (double-welded butt) longitudinal joint is fully radiographed. What joint efficiency E is used per Table UW-12?

Answer: 1.00
For a Type 1 joint with full radiography, Table UW-12 gives E = 1.00. Spot RT would give 0.85 and no RT 0.70.
Reference: ASME VIII-1 UW-12, Table UW-12
Question 10 · Calc — Pressure testing

A vessel has an MAWP of 200 psi. Assuming the test-temperature and design-temperature allowable stresses are equal, what is the standard hydrostatic test pressure per current ASME VIII-1 (UG-99(b))?

Answer: 260 psi
Standard hydrostatic test pressure = 1.3 × MAWP × (S_test/S_design). With the stress ratio = 1, = 1.3 × 200 = 260 psi. (The 1.5× factor is the older pre-1999 basis.)
Reference: ASME VIII-1 UG-99(b)

Question 11 · Calc — Weld sizing

A fillet weld has a required throat dimension of 0.35 in. Using the exam conversion factor of 0.7, what is the corresponding leg dimension?

Answer: 0.5 in
Throat = 0.7 × leg, so leg = throat / 0.7 = 0.35 / 0.7 = 0.5 in.
Reference: ASME VIII-1 UW-16 / API 510 BOK
Question 12 · API 510 — Responsibilities

Per API 510, who has overall responsibility for the in-service inspection program, including compliance with the code?

Answer: The owner/user of the equipment
API 510 assigns overall responsibility for the inspection, repair, alteration, and rerating program to the owner/user.
Reference: API 510 4.2
Question 13 · API 510 — Scope

API 510 applies primarily to which equipment?

Answer: In-service vessels: inspection, rating, repair, alteration
API 510 covers in-service pressure vessels — inspection, rating, repair, and alteration. New construction is covered by the construction code (e.g., ASME VIII).
Reference: API 510 1.1
Question 14 · API 510 — Rating

When establishing the MAWP for an existing pressure vessel, which thickness should be used in the calculation?

Answer: Actual thickness less corrosion before the next inspection
Rerating/MAWP determinations use the actual measured thickness minus the corrosion allowance that will be lost before the next inspection date.
Reference: API 510 7.5
Question 15 · API 510 — Rerating

A vessel may be rerated to a different MAWP only when which condition is met?

Answer: Calculations to the construction code, fully documented
Rerating requires calculations per the applicable construction code, documentation, and (typically) updating the nameplate; it is performed by or with the manufacturer/engineer per API 510.
Reference: API 510 8.1
Question 16 · API 510 — CMLs

Condition monitoring locations (CMLs) are best described as:

Answer: Designated areas monitored for damage and thickness
CMLs are designated areas on a vessel where inspections (often thickness readings) are taken to monitor the presence and rate of damage.
Reference: API 510 5.6
Question 17 · API 510 — Inspection types

An external inspection of an in-service vessel can typically be performed:

Answer: While the vessel remains on-stream
External inspections are generally on-stream activities; internal inspections require the vessel to be shut down and opened.
Reference: API 510 6.3
Question 18 · API 510 — Inspection intervals

The external inspection interval for most pressure vessels is not to exceed:

Answer: Not to exceed five years (or next internal)
API 510 generally limits external inspection intervals to 5 years (or the next internal/on-stream inspection, if sooner).
Reference: API 510 6.4
Question 19 · API 510 — RBI

When a risk-based inspection (RBI) assessment is used to set inspection intervals, API 510 requires that it be:

Answer: Documented and updated at the set interval
RBI assessments must be documented and reassessed/updated periodically and when conditions change (per API 510 referencing API 580).
Reference: API 510 6.2 / API 580
Question 20 · API 510 — Repairs/alterations

Which of the following is, by definition, an ALTERATION rather than a repair?

Answer: A change beyond the original design basis
An alteration changes the vessel from its original design (e.g., adding a nozzle, increasing MAWP). A repair restores the vessel to a safe operating condition without such a change.
Reference: API 510 3.1

API 510 exam FAQ

What does the API 510 exam cover?

The API 510 Pressure Vessel Inspector exam covers inspection, repair, alteration, and rerating of in-service pressure vessels, drawing on API 510 itself plus related documents in the published Body of Knowledge (including API RP 571, 572, 576, 577 and ASME code sections). Check api.org for the current Body of Knowledge and exam format.

How hard are the calculation questions?

Calculations — corrosion rate, remaining life, inspection intervals, required thickness, static head — are where most candidates win or lose points. They aren't conceptually hard, but you need the formulas to be automatic under time pressure. That's why this sample leans heavily on them.
