# CertPrep Practice

Static SEO landing pages with free sample questions, funneling to the App Store listings for
[API Inspector Cert Prep](https://apps.apple.com/us/app/api-inspector-cert-prep/id6785875538) and
[NDT Cert Study](https://apps.apple.com/us/app/ndt-cert-study/id6785204471).

- `gen.py` regenerates all pages from `api510_sample.json` / `ut2_sample.json`
  (samples drawn from the apps' SME-reviewed banks).
- Hosted on GitHub Pages from the `main` branch root. `.nojekyll` keeps Pages from
  running Jekyll.
